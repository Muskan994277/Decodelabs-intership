# 🤖 Rule-Based AI Chatbot

## 📌 Project Description
This is a Rule-Based AI Chatbot developed using Python. The chatbot responds to predefined user commands using if-elif-else conditions. It provides greetings, AI and Python information, current date and time, and other helpful responses through a simple command-line interface.

## ✨ Features
- 👋 Greeting Messages
- 🧠 AI Information
- 🐍 Python Information
- 💻 GitHub Information
- 🌐 LinkedIn Information
- 📅 Current Date
- 🕒 Current Time
- 📖 Help Command
- 🚪 Exit Command
- 🎨 Colorful Console Interface

## 🛠 Technologies Used
- Python 3
- Colorama
- Datetime

## 📂 Project Files
- chatbot.py
- responses.py
- banner.py
- colors.py
- utils.py
- README.md
- requirements.txt

## ▶️ How to Run
1. Install Python 3.
2. Install Colorama:
   ```
   pip install colorama
   ```
3. Run the project:
   ```
   python chatbot.py
   ```

## 📸 Output
The chatbot displays a colorful welcome banner and responds to predefined commands entered by the user.
